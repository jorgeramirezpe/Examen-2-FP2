PRÁCTICA FUNDAMENTOS DE LA PROGRAMACIÓN

1. ¿Que es POO?

POO significa Programación Orientada a Objetos y es una forma de programar utilizando objetos, propiedades, métodos, es la forma de programar más cercana a la vida real.

2. Explique la diferencia entre Clase y Objeto

Una clase es como un molde, es como la característica en sí, a diferencia de un Objeto, que mediante una instancia la clase recibe un valor y este se convierte en objeto.

3. ¿Que es Git y Github?

Git es un software de control de versiones mientras que Github es una plataforma para alojar proyectos utilizando el controlador de versiones Git.

3. ¿Que es Heroku?

Heroku es una plataforma para subir proyectos a la nube en distintos lenguajes de programación.
